$ insallasi XAMPP
    Unduh XAMPP:

    Kunjungi situs resmi XAMPP di https://www.apachefriends.org/
    Pilih versi XAMPP yang sesuai dengan sistem operasi Anda (Windows, macOS, Linux) dan versi PHP yang diinginkan.
    Klik tautan unduh yang tersedia untuk versi yang dipilih.
    Jalankan Instalasi XAMPP:
    
    Setelah unduhan selesai, buka file installer XAMPP yang telah Anda unduh.
    Ikuti petunjuk instalasi yang muncul di layar.
    Pilih komponen yang ingin Anda instal. Pastikan Anda memilih Apache, MySQL, dan PHPMyAdmin untuk menginstal phpMyAdmin.
    Tentukan Lokasi Instalasi:
    
    Pilih lokasi di mana Anda ingin menginstal XAMPP.
    Pada umumnya, direktori default (misalnya C:\xampp di Windows) sudah cukup sesuai.
    Selesai Instalasi:
    
    Setelah instalasi selesai, XAMPP akan terinstal di komputer Anda.
    Pastikan semua komponen (Apache, MySQL) berjalan dengan baik.
    Membuka phpMyAdmin:
    
    Buka XAMPP Control Panel melalui shortcut atau melalui direktori instalasi XAMPP.
    Start Apache dan MySQL dengan mengklik tombol Start di depan kedua komponen tersebut.
    Setelah Apache dan MySQL berjalan, klik tombol Admin di sebelah kanan MySQL pada XAMPP Control Panel. Ini akan membuka phpMyAdmin di browser default Anda.
